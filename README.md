# 🎮 R6 Tactical Callout Guide
### 競技射擊遊戲：視覺化戰術溝通工具
[![GitHub Pages](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://tyrfing2026.github.io/r6-callouts/)
[![Framework](https://img.shields.io/badge/Framework-Tailwind_CSS-blue?style=for-the-badge)](https://tailwindcss.com/)

這是一個專為《虹彩六號：圍攻行動》(Rainbow Six Siege) 玩家設計的**互動式地圖點位參考工具**。透過視覺化的界面，協助玩家在對戰環境下進行更精準的資訊傳達。

---

## 📌 專案核心動機 (Why this project?)
在《虹彩六號》的高強度對抗中，**「資訊不對稱」** 往往是導致戰鬥失敗的主因，我發現新手與業餘團隊在溝通時常面臨：
1. **認知負荷過重**：難以同時處理遊戲操作與記憶複雜的地圖名詞。
2. **檢索失敗**：在關鍵時刻無法從長期記憶中提取正確的點位名稱。

**解決方案**：開發此工具，利用視覺編碼（Visual Encoding）將空間位置與標準隊伍術語進行映射，降低溝通時的心理成本。

---

## ✨ 核心亮點 (Features)

* **📍 互動式地圖引導**：使用者點擊地圖區塊，即可即時顯示標準的中英文報點名稱。
* **📱 響應式佈局 (Responsive Design)**：使用 Tailwind CSS 打造，完美支援手機與平板。玩家可將其放在桌邊隨時參考。
* **⚡ 極速檢索**：輕量化架構，點開即用，確保在對戰空檔能快速查閱下一波段的戰術點位。
* **📂 地圖過濾系統**：快速切換不同競技場景（如：奧勒岡、俱樂部等），滿足不同戰術需求。

---

* **樣式設計**: **Tailwind CSS** (採用 Utility-first 理念進行快速開發與介面優化)。
* **版本控制**: Git / GitHub.
* **部署平台**: GitHub Pages.
